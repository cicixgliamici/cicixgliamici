## Selected Projects

### [Sophie](https://github.com/cicixgliamici/sophie)
A JVM-based **domain-specific language for trading strategies and portfolio allocation**, designed around a structured pipeline from parsing to execution.
- **What it shows:** language design, ANTLR-based parsing, typed ASTs, evaluator/execution pipeline, software architecture
- **Stack:** Scala 3, sbt, ANTLR4, ScalaTest

### [NASA Power of Ten Demo](https://github.com/cicixgliamici/nasa-power-of-ten-demo)
A compact C project inspired by the NASA/JPL *Power of Ten* rules for safety-critical software.
- **What it shows:** disciplined C programming, rule-to-implementation traceability, tests, engineering reflection
- **Stack:** C, Make, tests, technical documentation

### [Quantum-Ed](https://github.com/cicixgliamici/quantum_ed)
An educational repository for **quantum computing, quantum information, and hardware foundations**, combining theory notes, notebooks, code, and tests.
- **What it shows:** technical education design, mathematical clarity, scientific computing structure
- **Stack:** Python, NumPy, Jupyter, pytest

### [LeetCode](https://github.com/cicixgliamici/leetcode)
A repository of problem solutions with explanations, comments, and complexity-oriented practice material.
- **What it shows:** consistency, problem solving, interview-oriented algorithmic practice
- **Stack:** currently focused on structured multi-problem solutions and notes
